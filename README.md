Hello World in HTML

Description
This is simple HTML program

Files 
index.html -> this contains simple HTML code to display Hello World

Example Code
<!DOCTYPE html> 
<html> 
  <head>
    <title>Hello World</title> 
  </head> 
  <body> 
    <h1>Hello World!</h1>
  </body> 
</html>

How To Run-
1. Save the code in file named as index.html
2. Open the file in any browser(eg firefox, edge, etc)
3. The browser will display:
      Hello World !
