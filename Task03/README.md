# CSS Selectors

## Description
  Applied the basic properties of the CSS using Selectors

## Files 
  index.html - contains HTML code
  style.css - contains CSS properties

## Tag used in HTML
  h1 - presenting the head of the page
  h2 - other headings
  p - paragraphs
  li - 1. Ordered List
       2. Unordered List

## Selector used in CSS
  1. Element Selector
  2. ID Selector
  3. Class Selector

## How to run
  1. Save the code in a file named index.html.
  2. Open the file in any web browser (Chrome, Edge, Firefox, etc.).
  3. The browser will display output
