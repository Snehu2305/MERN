# Units, Box Models, Fonts

## Description
  These topics demonstrates the use of HTML and CSS viewport units to create a responsive webpage layout. The page contains an image and a button placed inside a container with
  equal spacing from the left and right sides and top and bottom spacing

## Features
  1. Responsive layout using viewport units(vw and vh)
  2. Gradient-styled button with customized font settings.
  3. Proper use of the CSS Box Model with box-sizing: border box.
  4. Clean and simple user interface.

## Technologies used
  HTML5
  CSS3

## Concepts Covered
  Viewport Units (vw, vh)
  Box Sizing
  Gradient Backgrounds
  Responsive Web Design
  CSS Styling and Layout
