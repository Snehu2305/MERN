# Laundry Services Hero Section

## Description
  In this webpage I've created navbar which contains Home, Services, About Us, Contact Us. At the left hand the logo is present and the right side username is present. The Under the navbar title and description about Service along with "Book a service today" button is present. Image of the Laundry also present.

## Features
  - Navbar for navigation
  - Online service book facility
  - Description about the Services
  - image representing the services
  - Proper use of display: inline-block, box-sizing: border-box
  - No Flexbox

## Technologies Used
  - HTML5
  - CSS3

## How to Run
  - Download or clone the project.
  - Open index.html in a web browser.
  - Output will be displayed.
