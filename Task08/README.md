# Laundry Service Responsive Website

## Overview
  This project is a responsive Laundry Service landing page built using HTML and CSS. It includes a navigation bar, hero section, call-to-action button, responsive design using viewport units, and a mobile-friendly hamburger menu created using CSS pseudo-classes.

## Features
  - Responsive Navigation Bar
  - Mobile Hamburger Menu
  - Hero Section with Attractive Laundry Image
  - Responsive Layout using Viewport Units (vw, vh)
  - Username Badge in Navigation Bar
  - Interactive Buttons with Hover Effects
  - Mobile-Friendly Design using Media Queries
  - CSS Pseudo Classes (:hover, :focus)
  - Clean and Modern User Interface

## Technologies Used
  - HTML5
  - CSS3

## Concepts Covered
  - Viewport Units (vw, vh)
  - Media Queries
  - Responsive Web Design
  - CSS Hover Effects
  - CSS Focus Pseudo Class
  - Inline Block Layout
  - CSS Positioning
  - Mobile Navigation Menu
  - Responsive Features
  
## Desktop View
  - Displays full navigation menu.
  - Hero content and image appear side by side.
  - Interactive call-to-action button.

## Mobile View
  - Navigation items are hidden.
  - Hamburger menu icon is displayed.
  - Menu items appear when the hamburger button receives focus.
  - Hero section stacks vertically for better readability.

## How to run
  1. Download or clone the project files.
  2. Ensure the following files are in the same folder:
            - task08.html
            - task08.css
  3. Open the project folder in Visual Studio Code.
  4. Install the Live Server extension (if not already installed).
  5. Right-click on task08.html.
  6. Select Open with Live Server.
  7. The website will open in your default browser.

  ### Alternative Method
      1. Locate task08.html in File Explorer.
      2. Double-click the file.
      3. It will open directly in your web browser
