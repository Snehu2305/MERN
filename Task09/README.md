# Transform in CSS

## Description
   Used the CSS 'transform' property to create interactive hover effects such as scaling and rotation for enhanced user experience.

## Feature
  - Used scale() transform for increasing the size of button
  - Used rotate() transform to tilt the button
  - It enhances the user experience

## Files
  - HTML5
    Contains HTML code for webpage
  - CSS3
    Contains styling code for webpage

## How to Run

1. Download or clone the project files.
2. Ensure the following files are in the same folder:

   * 'index.html'
   * 'style.css'
3. Open the project folder in Visual Studio Code.
4. Install the **Live Server** extension (if not already installed).
5. Right-click on 'task08.html'.
6. Select **Open with Live Server**.
7. The website will open in your default browser.

### Alternative Method

1. Locate 'index.html' in File Explorer.
2. Double-click the file.
3. It will open directly in your web browser.
