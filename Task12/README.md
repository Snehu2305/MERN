# Compound Interest Calculation

## Description
  A JavaScript program that calculates compound interest based on static values.

## program files
  - index.html
     Used <script> tag to link JavaScript file with html file

  - script.js
    - initialize and declared the variables.
    - Used compound interest formula.
    - Displayed the output using console.log

## how to run
  1. Download or clone the project.
  2. Keep 'index.html' and 'script.js' in the same folder.
  3. Open 'index.html' in a web browser.
  4. Use the webpage to test the JavaScript features.
