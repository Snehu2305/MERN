# Check if a Number is Even or Odd

## Description 
  A JavaScript program that checks whether a given number is even or odd.

## Program Files
  - index.html
      It connects JavaScript file to html code
  - script.js
     - It imports the evenOdd function from 'utility.js'
     - Checks the Number whether even or odd
  - utility.js
     - It contains export function evenOdd
     - Uses if-else statement to check whether the number is even or odd.
     - returns the output

## How to run
  1. Download or clone the project.
  2. Keep 'index.html' and 'script.js' in the same folder.
  3. Open 'index.html' in a web browser.
  4. Use the webpage to test the JavaScript features.
  5. Right click on webpage and click on 'inspect'
  6. Open console and see the output
