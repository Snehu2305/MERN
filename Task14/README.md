# JavaScript Loops

## Description
  JavaScript Program that performs various operations on a given number including finding sum of its digits, checking if it is an Armstrong number,
  checking if it is a prime number, and printing all its factors.

## Program Files
  - index.html
     - It connects the script.js file to html code.
  - script.js
     - It imports the functions from 'utility.js'.
     - Performs operations on the static variables.
  - utility.js
     - It contains export functions.
     - Each export function contains logic for the particular operation.
     - It returns result.

## How to Run
  1. Download or clone the project.
  2. Keep 'index.html', 'script.js' and 'utility.js' in the same folder.
  3. Open 'index.html' in a web browser.
  4. Use the webpage to test the JavaScript features.
  5. Right click on the webpage and click on 'inspect'.
  6. Click on console and see the output.
