# JavaScript - Arrays and Function

## Description
  JavaScript functions that receive an array as argument and perform various operations on it, including 
  finding the maximum number, calculating the sum of all elements, and counting the number of odd numbers.

## Program Files
  - index.html
      - It contains html code which connects JavaScript code with webpage
  - script.js
      - It contains JavaScript code.
      - Functions for finding the maximum number, calculating the sum of all elements, and counting the
        number of odd numbers.
      - It contains various types of functions like anonymous function, arrow function and named function.
      - Checks output using array

## How to Run
  1. Download or clone the project.
  2. Keep 'index.html' and 'script.js' in the same folder.
  3. Open 'index.html in a web browser.
  4. Use the webpage to test the JavaScript features.
  5. Right click on the webpage and click 'inspect'.
  6. Open console and check the output.
