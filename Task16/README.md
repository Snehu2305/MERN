# Map and Filter Exercise

# Description
  A webpage that displays student records using HTML and JavaScript. The web page includes some cards showing
student details such as name, marks, class and address. Additionally, a search box implemented to filter students by 
name dynamically.

## Features
  - cards with student info like student name, marks, class and address
  - Search box that filters student by first name.
  - Dynamically filters students
  - Attractive UI

## Program files
  - index.html
  - style.css
  - script.js

## How to Run
  1.Download or clone the project.
  2. Keep 'index.html' and 'script.js' in the same folder.
  3. Open 'index.html' in a web browser.
  4. Search the name to filter the student.
