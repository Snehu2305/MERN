# Promise-based Divide Function

## Description
  A JavaScript function that performs division using Promises. The function should handle cases such as 
division by zero and return the result or an appropriate error message.

## Working
  1. Takes two numbers as input and returns a Promise.
  2. Performs division of the first number by the second number.
  3. If the second number is zero, the function rejects the Promise with an appropriate error message.
  4. Otherwise, the function resolves the Promise with the result of the division.
  5. Uses dummy values for testing purposes.

## How to Run
  1. Download or clone the project.
  2. Keep 'index.html' and 'script.js' in the same folder.
  3. Open 'index.html' in a web browser.
  4. Use the webpage to test the JavaScript features.
  5. Right click on the webpage and select inspect.
  6. Select console and see the output.
