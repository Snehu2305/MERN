# JavaScript DOM and Event Handling

## Description
  A simple web page with interactive elements using HTML, CSS, and JavaScript. The web page will include
a header with greeting message, four boxes with different text named red, blue, green and yellow, an input
field to get the user's name, and a button to greet the user.

## Working
  1. Header initially displays text "Hello".
  2. Four boxes with text as red, blue, green and yellow respectively.
  3. Clicking on boxes the background changes to the text written on the boxes.
  4. Input given in input field and clicking on greet button will display "Hello input" in header.

## Program Files
  - index.html
  - style.css
  - script.js

## How to Run
  1.Download or clone the project.
  2. Keep 'index.html', 'style.css' and 'script.js' in the same folder.
  3. Open 'index.html' in a web browser.
  4. Web page will appear with header, 4 boxes, input field and button.
