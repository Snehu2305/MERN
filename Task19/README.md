# Service Booking Web App

## Description
  A web application that allows users to browse and book services.The application have a user-friendly
interface with a navigation bar, a section to display added items, and another section to browse and
select services. The user able to add services to their cart and book them.

## Feature
 - Logo on the left side of the navigation bar.
 - Includes navigation links for 'Home', 'Services', 'About Us' and 'Contact Us'.
 - Displays the username on the right side.
 - Implemented a red button for user logout.
 - Section which displays items added by th user.
 - Includes a dynamic list that gets updated when services are added from the right section.
 - Section which displays services.
 - Booking form for booking service.

## Program Files
  - index.html
  - style.css
  - script.js

## How to Run
  1. Download or clone the project.
  2. Keep 'index.html', 'style.css' and 'script.js' in the same folder.
3. Open 'index.html' in a web browser.
4. Use the webpage to test the JavaScript features.
