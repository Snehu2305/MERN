# Laundry Wallah HTML webpage

## Description 
A premier laundry service, is enhancing its online presence through a HTML webpage. The initial HTML code serves as a foundation for a user-friendly and visually appealing site. Web page is about creating a seamless digital experience, showcasing services, pricing, and easy navigation.

## Features
 1. Overview about the services provided by the Laundry Wallah
 2. Prices of the services
 3. Can book the service online

## Files 
index.html - contains HTML code for Laundry Wallah Webpage

## How to run
1. Save the code in file named as index.html
2. Open the file in any browser(eg firefox, edge, etc)
3. The browser will display:
      The webpage Laundry Wallah
